# 项目简介

SpringBoot Vue基层智能化人员调度系统是一款旨在提高人员调度效率的管理工具。该系统提供了一套完整的解决方案，以帮助企业或组织实现智能化的人员调度。

## 核心功能

- **系统首页**：提供系统概览，方便管理员快速了解系统状态。
- **人员管理**：包括人员基本信息、人员分组、人员状态等管理功能。
- **小组管理**：创建和管理不同的小组，以便于进行人员调度。
- **人员调用**：根据任务需求，智能分配人员到相应的小组。
- **物资信息**：管理物资的基本信息，包括物资名称、类型、数量等。
- **物资入库**：记录物资入库信息，包括入库时间、数量等。
- **物资调用**：根据任务需求，分配物资给相应的小组。
- **物资保养**：记录物资的保养情况，确保物资的正常使用。
- **个人信息**：用户可以查看和修改自己的个人信息。

## 适用场景

- **企业人力资源管理部门**：用于优化人员配置，提高工作效率。
- **政府部门**：用于公共资源调度，提高服务效率。
- **其他组织**：适用于任何需要进行人员调度和资源管理的场景。

## 技术特点

- **B/S架构**：基于浏览器/服务器架构，易于部署和使用。
- **MVC模式**：采用模型-视图-控制器模式，提高代码的可维护性和扩展性。
- **Java技术栈**：使用Java作为开发语言，保证了系统的稳定性和性能。
- **SpringBoot框架**：简化开发流程，提高开发效率。
- **Vue框架**：提供丰富的前端功能，提升用户体验。
